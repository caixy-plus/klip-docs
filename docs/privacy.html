Klip 隐私政策

最后更新：2026年7月7日 · English

Klip（「本应用」）由蔡新云（Cai Xinyun）开发，是一款 macOS 菜单栏剪贴板历史管理器。

数据收集

本应用不收集、不上传、不与任何第三方共享您的个人数据。本应用不包含任何网络功能、第三方分析 SDK 或崩溃上报服务。

本地存储的数据

  • 剪贴板历史：您复制的文本、图片及文件引用仅存储在您的 Mac 本机，用于在应用内展示历史记录与搜索。
  • 数据库加密：历史记录数据库使用 SQLCipher 加密，加密密钥保存在系统钥匙串（Keychain）中。
  • 应用设置：如热键、保留策略、排除应用列表等偏好设置存储在系统 UserDefaults 中。
  • 图片文件：复制的图片以文件形式保存在本机 Application Support 目录下。

以上数据均仅在您的设备上处理，不会离开您的 Mac。

权限

本应用运行在 macOS 沙盒中，不请求辅助功能、网络、位置或通讯录等系统权限。粘贴采用「复制模式」：将内容写回系统剪贴板后把焦点还给原应用，由您自行按 ⌘V 完成粘贴。

您的控制

  • 随时在「设置 → 隐私」中暂停记录或锁屏时暂停记录。
  • 排除特定应用，使其复制内容不被记录。
  • 随时清空历史记录（固定项可保留）。
  • 卸载应用即可删除本机上的全部相关数据。

儿童隐私

本应用不面向 13 岁以下儿童，也不会故意收集儿童信息。
