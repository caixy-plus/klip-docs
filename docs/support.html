常见问题

如何唤起历史面板？
默认快捷键为 ⌘K，可在「设置 → 通用」中修改。

如何粘贴历史记录？
在面板中选中条目后按 ⏎，内容会写入系统剪贴板并关闭面板；回到原应用后按 ⌘V 粘贴。

如何暂停记录剪贴板？
打开「设置 → 隐私」，开启「暂停记录」或「锁屏时暂停记录」。

如何排除某个应用？
在「设置 → 隐私 → 已排除的应用」中点击「添加应用」，选择不希望被记录的应用。

数据存储在哪里？
所有数据仅保存在本机：~/Library/Application Support/Klip/。数据库经 SQLCipher 加密，密钥在系统钥匙串中。本应用无任何网络连接，数据不会上传。

如何清空历史？
「设置 → 通用 → 清空历史（保留固定项）」。固定的条目不会被删除。
